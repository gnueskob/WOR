  • PUT /user/login

hive_id STRING Y 하이브 ID

회원가입

  • POST /user/register

hive_id STRING Y 하이브 ID
hive_uid INT Y 하이브 UID
country STRING Y 국가
lang STRING Y 언어
os_version string Y OS 정보
app_version string Y 앱 버전

기획 데이터 관련

기획데이터 업로드

  • POST /plan/upload
요청변수 타입 필수 여부 설명
territory CSV File N 영토 정보
tile CSV File N 영내 정보
resources CSV File N 자원 정보
building CSV File N 건물 정보
weapon CSV File N 무기 정보
buff CSV File N 버프 정보
boss CSV File N 보스 정보
trophy CSV File N 전리품 정보
unit CSV File N 월드 단위 데이터 정보
